Noise.js

Noise.js addes animation to your typography like a computer bug.
this program uses css3 animation.

Example

animation trigger is mouse hovering.

Usage

Setup

You don't need to load jQuery...
Add below script into your head tag.

<script src="./file/to/path/noise.js"></script>

Config

<script>noise('your_class_name');</script>
